# 🦷 BrightSmile Lead Qualification & Automation Workflow

An advanced, production-ready automation workflow built on **n8n** that intercepts dental appointment inquiries from the BrightSmile web frontend, sanitizes and validates contact payloads via JavaScript, and routes clean data to Google Sheets while handling dynamic customer communications.

---

## 🚀 Overview

This workflow acts as the backend automation engine for BrightSmile. It intercepts incoming frontend webhooks, applies strict lead-qualification algorithms to filter out fake or poorly formatted data, saves high-fidelity records to Google Sheets, and automates customized email confirmations.
